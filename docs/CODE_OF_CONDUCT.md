# Code of Conduct

## Our Pledge

We pledge to make participation in this project a respectful, harassment-free experience for everyone.

## Expected Behavior

- Be respectful and direct.
- Focus technical disagreement on evidence, tradeoffs, and project goals.
- Give and receive feedback constructively.
- Avoid sharing private information, credentials, or sensitive operational details.

## Unacceptable Behavior

- Harassment, threats, insults, or discriminatory language.
- Publishing private information without consent.
- Deliberately disrupting discussions, issues, pull requests, or maintainer work.
- Attempting to use the project to bypass provider terms, scrape credentials, or share unauthorized access.

## Enforcement

Maintainers may remove comments, close issues, reject pull requests, or block participants who violate this code of conduct. Serious security or abuse concerns may be handled privately.
